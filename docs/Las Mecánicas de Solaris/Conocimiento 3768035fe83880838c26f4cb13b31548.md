# Conocimiento

Sirve para conseguir información sobre el mundo y las personas y criaturas que lo habitan. Permite conseguir bonificadores y reducir la dificultad de las tiradas para conocer puntos débiles y entender lo que te rodea.

Hay 4 grados de conocimiento

Condición: hablar con alguien

- Rumor
- Pista
- Evidencia
- Verdad

## Etiquetas

Cada conocimiento viene asociado con una etiqueta, que puede ser genérica o específica.

Una etiqueta genérica indica el ámbito de ese conocimiento. Esta etiqueta, se puede convertir en una etiqueta específica. Por ejemplo, una conocimiento con la etiqueta de Persona, se puede transformar a una etiqueta de una persona en específico. 

Lista de etiquetas:

- Persona
- Ubicación
- Objeto
- Historia
- Natural
- Futura
- Universal (Comodín)
