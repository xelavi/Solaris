# Activas

### Body

**Carga** → Ataque a distancia, gastas 2 acciones en correr + Ataque

**Tensión** → (Reacción) +3 a Resistencia

**Adrenalina** → (Mental) +1 BODY hasta próximo turno, una vez por combate.

**Interceptar** → (Reacción) A una criatura adyacente puedes recibir el daño de tu compañero pusilánime.

**Cura →** (Mental) Te curas tu regeneración en PV. Una vez por día.

**Ataque pesado** → Esta activa no se puede usar con ataques a distancia o un arma de categoria Ligera. El personaje usa dos acciones para realizar un ataque pesado, duplicando el daño normal del ataque. En el caso de que el arma usada sea de categoria Pesada, se usaran 3 acciones en vez de 2 y el daño se triplicará.

### Agilidad

**Apuntar** → (Mental) 2 Acciones sumar Deadeye ToHit (Concentración)

**Riposte** → (Reacción) Suma tu nivel a tu Esquiva contra ese ataque específico.

**Ataque de oportunidad** →(Reacción) Si alguien sale de tu rango de ataque cuerpo a cuerpo, le puedes atacar

**Contraataque** → (Reacción) Puedes atacar si te atacan

### Mind

**Procesar** → Permite realizar acciones mentales que de normal costarían 1 accion son gratuitas.

**Instruir** → -Gastas 1 accion para darle ventaja 1 a un aliado.
